## Getting Started

### Prerequisites

- Node.js 20+
- pnpm

### Run Locally

```bash
# 1. Clone repository
git clone https://github.com/clroot/selah-web-application.git
cd selah-web-application

# 2. Install dependencies
pnpm install

# 3. Run development server
pnpm dev

# 4. Access application
open http://localhost:3000
```

### Build

```bash
# Build
pnpm build

# Start production server
pnpm start

# Lint & Format
pnpm lint
pnpm format
```
